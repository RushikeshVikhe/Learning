How to push and pull image from ECR using publice repository
Create IAM User 
Create Access Key

Install AWS CLI v2:
curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
sudo apt install unzip
unzip awscliv2.zip
sudo ./aws/install -i /usr/local/aws-cli -b /usr/local/bin --update
aws configure   .....paste access key and secret access key


Create public repository in ECR go inside you will see "View push commands" copy and paste that

To pull image from ECR cmd is - docker pull <url>
